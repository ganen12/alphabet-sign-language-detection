Alphabet Level Sign Language Project
===================================

INSTRUKSI
-------------------

1. Persiapkan Virtual Environment
   - Pastikan Python terinstal di komputer.
   - Buka terminal/command prompt di folder proyek:
     
     cd "alphabet level sign language 2"

   - Buat virtual environment:
     
     python -m venv mp_env

   - Aktifkan virtual environment:
     * Windows:
       mp_env\Scripts\activate

2. Install Library yang Dibutuhkan
   - Jalankan perintah berikut untuk menginstall semua library:     
     pip install mediapipe==0.10.14 protobuf==4.25.3 tqdm scikit-learn opencv-python

3. Struktur Folder (abaikan)
   - ASL_Dataset/
       Train/  --> data latihan
       Test/   --> data testing
   - mp_env/  --> environment project
   - model.p  --> model machine learning yang sudah dilatih
   - train_data.pickle, test_data.pickle  --> dataset yang sudah diproses
   - test.ipynb --> file utama untuk menjalankan deteksi dan prediksi ASL

4. Menjalankan Project
   - Pastikan virtual environment sudah aktif.
   - Buka file `test.ipynb` menggunakan Jupyter Notebook atau Visual Studio Code:
     
     Jika dengan VSCode, ganti kernel ke kernel python virtual env (mp_env)

   - Jalankan sel di notebook sesuai urutan:
     1. 'Import library' dan 'Init Awal'
     2. 'Deteksi Tangan Realtime'
     3. Pastikan webcam berfungsi untuk mendeteksi gestur tangan
     4. Tekan tombol 'q' untuk keluar dari deteksi

5. Menonaktifkan Virtual Environment
   - Gunakan perintah ini di command:
     >'deactivate'
